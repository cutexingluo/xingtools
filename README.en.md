# java-cn-sdk XingTools springboot 分组工具包

#### Description
星天制作的 Java sdk 工具包 分组版本。
它是一个整合各工具类的整合starter。具体详情详见 https://gitee.com/SangonomiyaKokomi/xingtool 这个原sdk。
该分组工具包为升级版，将核心拆分应对不同使用情况，大大提高灵活性，未来原sdk可能会依赖该升级版的sdk。如果你是jdk任何版本都可以进行使用。唯一缺点就是可能会需要两个及以上的依赖。
